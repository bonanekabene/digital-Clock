## Features

1. **Digital Clock**  
   Displays current time in HH:MM:SS format with large, clear typography.

2. **Alarm**  
   Set and clear alarms directly from the settings menu. Triggers an alert and audio when the alarm time matches the current time.

3. **Audio Playback**  
   Play/pause a clock-themed audio track.

4. **Background Switcher**  
   Cycle through beautiful background images with a single button.

5. **Weather Section** *(placeholder)*  
   Shows weather information (city, temperature, condition). Can be integrated with APIs like OpenWeatherMap.

6. **Responsive Design**  
   Works on desktops, tablets, and small mobile devices (down to 400px width).
